# Starlords - منصة متعددة الأغراض

🌟 منصة احترافية لبث الأنمي مع قائمة مهام ولوحة طقس وموّلد نكات

## ✨ المميزات

- 📋 **قائمة المهام** - إدارة المهام مع الأولويات والتاريخ
- 🌤️ **لوحة الطقس** - بيانات الطقس الحالية والتوقعات
- 😄 **موّلد النكات** - نكات عشوائية مع صوت
- 🌙 **وضع ليلي** - دعم كامل للوضع الليلي
- 🇸🇦 **عربي كامل** - RTL واجهة عربية
- 💾 **تخزين محلي** - حفظ البيانات بأمان
- ⚡ **أداء عالي** - React 18 + Vite

## 🛠️ التكنولوجيات

- React 18.2.0
- TypeScript 5.3
- Tailwind CSS 3.4
- Vite 5.0
- Zustand (State Management)
- React Router 6.20

## 🚀 البدء

```bash
# تثبيت المتطلبات
npm install

# تشغيل خادم التطوير
npm run dev

# بناء الإنتاج
npm run build

# معاينة الإنتاج
npm run preview
```

## 📱 الصفحات

- `/` - الصفحة الرئيسية
- `/todo` - قائمة المهام
- `/weather` - لوحة الطقس
- `/jokes` - موّلد النكات

## 🔑 متغيرات البيئة

```env
VITE_OPENWEATHER_API_KEY=your-api-key
```
