# Starlords - منصة بث الأنمي

🌟 منصة احترافية لبث الأنمي مع لوحة تحكم إدارية ودعم كامل للعربية

## ✨ الميزات

- 📱 تطبيق ويب متقدم (PWA)
- 🌍 دعم كامل للعربية والإنجليزية مع RTL
- 🌓 وضع ليلي ومضيء
- ⚡ أداء عالي جداً
- 🎨 واجهة مستخدم احترافية
- 🔐 نظام مصادقة آمن
- 📊 لوحة تحكم إدارية متقدمة
- 📺 مشغل فيديو مع خيارات متعددة

## 🛠️ التكنولوجيا

- **React 19** - مع أحدث الميزات
- **TypeScript** - للأمان النوعي
- **Tailwind CSS** - لتصميم احترافي
- **Vite** - لبناء سريع
- **Zustand** - لإدارة الحالة
- **React Router v7** - للتوجيه
- **Supabase** - للمتطلبات الخلفية
- **Framer Motion** - للرسوم المتحركة

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+
- npm أو yarn

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/vash007-1/starlords-.git
cd starlords-

# تثبيت المتطلبات
npm install

# تشغيل خادم التطوير
npm run dev
```

### البناء للإنتاج

```bash
npm run build
npm run preview
```
